# 字趣 App


## 产品简介

"字趣"是一款面向 6~10 岁儿童的趣味汉字书写练习 App，主要使用场景为 iPad + Apple Pencil 触控笔。核心理念是"让写字变成一场冒险"，通过笔顺动画演示、跟写测验和星星奖励机制，激发孩子主动练字的兴趣。

**目标用户**：
- 主要用户：6~10 岁儿童（操作练字功能）
- 次要用户：家长（配置练习内容）

**产品调性**：温暖、童趣但不幼稚，鼓励而非说教。参考 Duolingo、洪恩识字的设计语言。

**吉祥物**：一只戴着小帽子、手持毛笔的橙色小狐狸。

**UI 要求**：
- 所有可交互元素必须有足够大的触控区域，适合儿童手指和触控笔操作
- 适配 iPad 横屏和竖屏
- 界面简洁，留白充足，避免信息过载
- 文字大小要适合儿童阅读

---

## 页面 1 — 首页

App 的入口页面，给孩子营造欢迎和成就感。

**功能元素**：
- 显示累计获得的星星总数
- 显示连续练习天数
- 设置入口（进入家长设置页）
- 吉祥物形象展示
- App 名称"字趣"和标语"让写字变成一场冒险"
- 三项统计数据：待练习字数、已掌握字数、总星星数
- 主操作按钮："开始练字"（进入练字页）
- 次要入口："设置练习的字"（进入设置页）

---

## 页面 2 — 练字页（核心页面）

孩子练习写字的主要场景，包含 3 个状态流转：预览 → 书写 → 结果。

### 通用元素（贯穿所有状态）
- 返回首页按钮
- 当前星星数显示
- 练习进度条（如"进度 3/10"及百分比）

### 状态 A — 预览态
孩子可以先观察字形和笔顺动画，再决定开始书写。

- 当前要练习的汉字大号展示
- 田字格书写区域（白底 + 十字线和对角辅助线），格内显示完整汉字
- 操作按钮："看笔顺"（播放笔顺动画）和"开始写"（进入书写状态）

### 状态 B — 书写态
孩子在田字格上跟着笔顺写字，系统实时检测每一笔。

- 提示标签"跟着写吧！"
- 错误计数显示（如"错误 2 次"）
- 笔画进度指示器：一排小圆点，已完成的笔画和未完成的笔画用不同视觉状态区分
- 操作按钮："重新开始"

### 状态 C — 结果态
书写完成后展示评价，给予正向激励。

- 星星评价展示（最高 3 颗星），带动画效果
- 鼓励文字：
  - 3 星："太棒了！完美！"
  - 2 星："很不错！继续加油！"
  - 1 星："加油，再来一次！"
- 显示获得星数和错误次数
- 操作按钮："再写一次"和"下一个"（最后一个字时变为"完成"）
- 满分（3 星）时触发撒花庆祝效果

## 页面 3 — 设置页（家长操作）

供家长配置孩子要练习的汉字列表，操作风格比练字页更工具化，适合成人使用。

**输入区**：
- 标题"输入要练习的汉字"
- 说明文字提示"可一次输入多个汉字，如'春天花开'，系统会自动拆分"
- 文本输入框 + 添加按钮

**快捷预设区**：
- 标题"快速添加常用字"
- 3 个预设按钮："+ 一年级上"、"+ 一年级下"、"+ 常用字"，点击后将对应字组批量添加到列表

**字表展示区**：
- 标题"当前练习列表"及字数统计
- 重置（恢复默认）和清空按钮
- 网格排列的汉字卡片，每个卡片显示一个汉字
- 已掌握的汉字卡片上有星星角标
- 每个卡片可单独删除

**页面级操作**：
- 返回按钮
- 保存按钮（有未保存修改时可用，无修改时禁用）
